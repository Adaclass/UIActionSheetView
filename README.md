#UIActionView
--------------------------
####一、简介：

一个仿iOS ActionView，支持文本及颜色设置

**1.1 Gradle集成**

```
allprojects {
    repositories {
        ...
        maven { url "https://jitpack.io" }
    }
}
```

```
dependencies {
     compile 'com.github.AriesHoo:UIActionSheetView:1.0.1'
}
```
